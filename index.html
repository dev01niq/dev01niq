Featured Projects

AI Reservation & Review Automation Platform

As a Founding Engineer at Troy AI, I built an AI-powered platform from scratch to help restaurants automate customer engagement, review management, and reservation workflows.

Challenge:

Large-scale review ingestion during onboarding caused performance bottlenecks due to synchronous processing.

Solution & Impact:

Redesigned the system into an event-driven architecture using AWS Lambda with retries and dead-letter queues. This reduced onboarding latency and dramatically improved scalability and reliability for future AI workflows.

React Python (FastAPI) AWS Lambda OpenAI

Healthcare Identity & Data Workspace System

As a Full-Stack Engineer, I built microservices for healthcare data onboarding, identity management, and secure analytics environments on GCP.

Challenge:

Clients needed a secure, isolated environment to analyze purchased healthcare datasets, and the platform lacked a reliable way to restore individual database tables after failed migrations.

Solution & Impact:

I built an automated VM provisioning workflow with dynamic Jupyter environments and role-based dataset access. I also engineered a custom table-level backup and restore pipeline, which improved migration safety and significantly reduced recovery time.

React Python (Django) GCP IAM
